# radiclone-fishin
A clone of a flash game called "Radical Fishin" by "Vlambeer" from 2010 using the Godot game engine.

DISCLAIMER: This is a non-commercial, educational clone of the classic 2010 Flash game *Radical Fishing* by *Vlambeer*. Built from scratch using the Godot game engine. The primary goal of this project was to learn fundamentals of game development, specifically in Godot, by imitating the core gameplay of an existing game. All code and assets are original.
